Rules

• To hit with a weapon, roll 1d20 >= 1/2 of target's ARMR
• To land a spell, roll 1d20 >= 1/2 of target's MAGI
• Spells cost 1d3 MAGI on cast
• When HLTH = 0, roll 1d20 >= 10 each turn to stay alive
• Advantage / Disadvantage = Roll twice and take highest / lowest
• To move in stealth, roll 1d20 >= 10
• All other checks must roll 1d20 >= respective stat
• Round up on all divided values
• 1d3 = 1d6 / 2

• To set up initiative, order all participants by DEXT
• Sessions should last from ~30-60 minutes (possibly longer)
• Sessions should contain ~1-2 simple combat sequences
• Use these same rules to create NPCs or enemies
