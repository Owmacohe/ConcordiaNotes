Here's a Dandân!

A little while ago, I heard about the format Forgetful Fish. It's a 1v1 format that uses a shared deck, shared graveyard, library manipulation, bouncing, and so very many mindgames. It seems like a lot of fun, and very quick too.

Oh yeah, and each 80 card deck contains 10 copies of Dandân and 8 copies of Memory Lapse.

Forgetful Fish has of course spawned countless variants.
Mono red versions, tempo versions, Izzet versions.

So I thought to myself: How can I make something similar?

Which card could I build an entire format around? What card is its own engine?

BEHOLD: Life From The Loam

At times, the great horror of Modern Dredge decks. At its peak, a single copy would cost you $30 USD. Let's make a deck with 8 of them.

In a 1v1 game there are going to be a lot of cards drawn. That's a lot of Dredging. That's a lot of milling. Very quickly, we'll end up with a library that's been flipped on its head. If that's the case, we don't need to care about life anymore. In fact, we won't even need any Creatures.

So it's a weird backwards race then? Let's speed up that race.

Oh, and we're only playing with 40 cards.
